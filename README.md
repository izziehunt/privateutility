# 🌊 Waypoint - Travel App Prototype

An iPhone 17 app prototype featuring an underwater/ocean theme with interactive sea animals and travel planning features.

## ✨ Features

- **10-Second Animated Splash Screen** - Wave pattern that drifts smoothly from side to side
- **Interactive Map** - Animated sea creatures (octopuses, shrimp, crab) with flowing wave animations
- **Destination Details** - Click luggage tags to view detailed information about locations (Boston)
- **Captain's Log (Travel Journal)** - Document your travels with a nautical/maritime aesthetic
- **Character Customization** - Customize your traveler avatar with 16 hairstyles, 8 hair colors, 10 skin tones, and various outfit options
- **iPhone 17 Mockup** - Realistic device frame with Dynamic Island styling

## 🎨 Design Theme

- Ocean/underwater aesthetic with cyan and blue gradients
- Maritime terminology throughout
- Animated sea creatures (octopus, shrimp, crab)
- Wave patterns and floating bubbles
- Nautical UI elements (anchors, compass, etc.)

## 🛠️ Technologies

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS v4** - Styling
- **Motion (Framer Motion)** - Animations
- **Lucide React** - Icons
- **Figma** - Design imports

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ and pnpm

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## 📱 Components

- **App.tsx** - iPhone 17 mockup wrapper
- **SplashScreen.tsx** - Main coordinator component
- **MapScreen** - Interactive map with animated sea creatures
- **BostonDetails** - Sliding destination detail panel
- **TravelJournal** - Captain's Log journal feature
